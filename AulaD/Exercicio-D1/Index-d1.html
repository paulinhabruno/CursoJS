Anotações

  1. DOM = Document Object Model. Conjunto de objetos dentro do navegador que vai dar acesso aos componentes internos do website.
  2. DOM não funciona dentro do node.
  3. DOM está presente quando está rodando JS dentro do navegador.
  4. Tipos de seleção:
    1. Selecionando por marca: documente.getElementsByTagName('item selecionado')[entra a posição do item, caso tenha mais de 1, começando por 0]
      1. Pode mudar o style
      2. innerText = mostra o resultado no navegador sem os comandos/tags filhos.
      3. innerHTML = Mostra o resultado formatado/com os comandos filhos.
    2. Selecionando por ID: document.getElementById()
    3. Selecionando por Name: document.getElementByName()
    4. Selecionando por Classe: document.getElementByClass()
    5. Selecionando por Selector: document.querySelector('coloca o seletor/tag')
