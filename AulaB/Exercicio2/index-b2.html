Anotações

Dicas

  1. Com o sinal de = entende-se "recebe" em JS
  2. O sinal de + significa que está concatenando (quando é string + string), mas também como adição(soma) (quando é number + number).
  3. Janelas alert retornam string
  4. .length: Para ver o tamanho da string.
  5. .toUpperCase(): Para colocar com letras maiúsculos
  6. .toLowerCase() : joga tudo para letras minúsculas.

Para converter para number

  1. Number.parseInt(n) para converter para inteiro.
  2. Number.parseFloat(n) para converter para números decimais.
  3. Number sozinho também pode ser usado. Nesse caso o JS vai verificar qual o tipo de number usado, se é int ou float.

Para converter para string

  1. String(n)
  2. n.toString

Para formatar a string

  1. Concatenando: 'Eu estou estudando' + s (quando s for uma variável de texto)
  2. TemplateString usando crase: `Eu estou aprendendo ${s}`
  3. Na interpolação com placeholder usa-se crase.

    4. O cifrão ${} chama-se placeholder.
